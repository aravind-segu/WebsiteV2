Munch

Top 5 and Wolfram Alpha Award

Munch is a cross-platform mobile app, which allows users to learn the nutritional information of their food with a quick picture. Munch utilizes a machine learning and artificial intelligence library, Clarifai to perform image recognition and identify the food snapped by the user. Once the photo is retrieved a node server, hosted on AWS, utilizes the image information to query a nutritional database and retrieve all the necessary information about the food. This information is then stored and the user can keep track of their food intake using various informative graphs. The app was built using Ionic, in order to serve both Iphone and Android users.

Checkr

CheckR is a cross platform mobile app which uses Optical Character Recognition in order to find spelling mistakes in handwriting. Once a user takes a pictures of their notes with the camera, CheckR uses Microsoft Azure Machine Learning API in order to recognize words from the image. Once recognized these words are checked in the dictionary and the misspelled words are alerted to the user. The mobile app for CheckR was built using Ionic as it will be able to serve both IOS and Android users.

Findr

MLH 3rd Place

FindR is an autonomous self-driving robot that is designed to help people in distress in times of emergencies. FindR aims to help firefighters by implementing advanced people recognition. With robust self-driving capabilities this robot can navigate through corridors and search rooms for people. If found, the FindR sends the GPS Co-ordinates along with pictures to a web interface for emergency crews to analyze the situation and respond accordingly. Additionally, FindR paths its route allowing firefighters to follow after it in areas of heavy smoke or dark areas. The robot was built with a Raspberry-Pi and utilizes OpenCV and Firebase to recognize humans and send real time updates.

DriveSafe

Python

DriveSafe is a three layer neural network built in Python to identify impaired driving. Trained with multiple dashboard camera images from the Kaggle Database, this neural network performs image recognition to keep drivers safe on the road. The neural Network utilizes PCA and conjugate graident algorithms to improve the rate of training and uses regularization to preven overfitting.

Jump

C++, Energia, Tiva C

JUMP is an interactive and fun-filled multi-level game where the main objective is to reach the end of the level by dodging obstacles. This game was built on a Tiva C Board with an Orbit Booster Pack add-on. Coded in Energia, this program uses a mixture of C, C++ and Tiva C Board functions. The human interaction of the game is developed using an LED Screen, Potentiometer, Accelerometer and Buttons. The major features of the game include its ability to create a new level on every play to keep the user engaged and the use of powerups to help finish a particular level. The game also utilizes the EEPROM on the Board to save the high scores.
